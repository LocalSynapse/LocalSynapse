## What is LocalSynapse?

LocalSynapse is a **local file search engine** that finds files by name, content, and meaning — 100% offline.

### 3-Tier Search
- **Filename Search** — Instant search across all drives
- **Full-Text Search** — Search inside document contents (BM25)
- **Semantic Search** — AI-powered meaning-based search (BGE-M3)

### Supported File Types
| Category | Formats |
|----------|---------|
| Documents | PDF, DOCX, XLSX, PPTX, TXT, MD, CSV, JSON, LOG |
| Korean | HWP, HWPX |
| Email | EML, MSG |

### Privacy First
- 100% offline operation
- No telemetry, no analytics
- No cloud sync
- No login required
- All data stays on your device

### System Requirements
- Windows 10/11 (x64)
- RAM: 4GB minimum (8GB recommended)
- Disk: ~150MB (app) + ~2.3GB (AI model, optional)
