Drag LocalSynapse to Applications to install
!
First launch on macOS
macOS may block this app because it is not signed with an Apple certificate.
This is a security warning, not a problem with the app.
A
Right-click to open
1
Control + click LocalSynapse in Applications
2
Click "Open" in the menu
3
Click "Open" again to confirm
B
Terminal command
Open Terminal and paste:
xattr -cr /Applications/LocalSynapse.app
Then double-click the app to launch.
You only need to do this once. Subsequent launches will work normally.
